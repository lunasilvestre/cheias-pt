# Temporal Data Backbone

Full crisis window dataset: **Dec 1, 2025 → Feb 15, 2026** (77 days).
Generated by `scripts/fetch_*.py` and `scripts/compute_precondition.py`.

## Datasets

### 1. SST Anomaly (`sst/`)
- **Source:** NOAA OISST v2.1, 0.25° daily grid
- **Area:** North Atlantic moisture corridor (60°W–5°E, 20°N–60°N)
- **Coverage:** 62 days (Dec 1 – Jan 31; Feb not yet published by NOAA)
- **Format:** 62 individual COG files (`sst/daily/*.tif`) + combined NetCDF (`sst_anomaly.nc`, 9.9 MB)
- **Shape:** 160 × 260 pixels per day
- **Range:** -8.7°C to +9.3°C anomaly
- **Chapter:** 2 (The Atlantic Engine)

### 2. Soil Moisture (`moisture/`)
- **Source:** Open-Meteo Historical API (ERA5-Land), ~11km resolution
- **Variables:** `sm_0_7`, `sm_7_28`, `sm_28_100`, `sm_rootzone` (depth-weighted)
- **Grid:** 342 points across Portugal (0.25° spacing)
- **Coverage:** 77 days, 26,334 records
- **Root-zone range:** 0.000–0.515 m³/m³
- **Format:** Parquet (0.2 MB)
- **Chapters:** 3, 5, 7

### 3. Precipitation (`precipitation/`)
- **Source:** Open-Meteo Historical API
- **Variables:** `precip_mm`, `rain_mm`, rolling accumulations (`precip_3d`, `7d`, `14d`, `30d`)
- **Grid:** 342 points (same as soil moisture)
- **Coverage:** 77 days, 26,334 records
- **Max daily:** 109.8 mm | Max 7-day: 353.5 mm
- **Format:** Parquet (0.3 MB)
- **Chapters:** 3, 4, 7

### 4. River Discharge (`discharge/`)
- **Source:** Open-Meteo Flood API (GloFAS proxy), 5km grid
- **Stations:** 11 points across 9 basins (Tejo×2, Douro×2, Sado, Mondego, Guadiana, Minho, Vouga, Lis, Zêzere)
- **Variables:** `discharge`, `discharge_median`, `discharge_max`, `discharge_min`, `discharge_ratio`
- **Coverage:** 77 days, 847 records
- **Format:** Parquet (32 KB)
- **Chapters:** 4, 5
- **Note:** Discharge ratios (1.1–1.6×) appear modest because GloFAS climatological median is already elevated in Jan-Feb wet season. Use absolute values or within-season amplification for narrative.

### 5. IVT Proxy (`ivt/`)
- **Source:** Open-Meteo Historical API (surface variables as atmospheric river proxy)
- **Variables:** `moisture_flux` (RH × wind_speed × directional_weight toward NE), `rh_mean`, `wind_mean`, `precip_sum`
- **Grid:** 1,495 points across North Atlantic (1° spacing, 25°N–55°N, 45°W–5°E)
- **Coverage:** 77 days, 115,115 records
- **Format:** Parquet (1.5 MB)
- **Chapter:** 2 (The Atlantic Engine)
- **Note:** True IVT requires pressure-level specific humidity (unavailable in Open-Meteo). This surface proxy captures the same SW→NE moisture corridor pattern.

### 6. Precondition Index (`precondition/`)
- **Source:** Computed from soil moisture + precipitation
- **Formula:** `0.6 × (precip_3d / remaining_capacity) + 0.4 × (precip_14d / p90)`
- **Classification:** green (<0.3), yellow (0.3–0.6), orange (0.6–0.9), red (>0.9)
- **Grid:** 342 points, 77 days, 26,334 records
- **Format:** Parquet (1.1 MB)
- **Chapters:** 5, 7, 8

## Regeneration

```bash
source .venv/bin/activate
pip install xarray netcdf4 rioxarray numpy pandas requests tqdm pyarrow

# Run in order (datasets 1-4 are independent, 6 depends on 2+3)
python3 scripts/fetch_discharge.py        # ~1 min
python3 scripts/fetch_sst.py              # ~5 min
python3 scripts/fetch_soil_precip.py      # ~15 min
python3 scripts/fetch_ivt_proxy.py        # ~10 min
python3 scripts/compute_precondition.py   # ~30 sec
python3 scripts/validate_temporal.py      # validation
```

## Data Gaps

- **SST Feb 1–15:** NOAA OISST typically has a ~2 week publication lag. Re-run `fetch_sst.py` later to fill.
- **3 soil/precip grid points:** Timed out (37.25°/-6.25°, 41.0°/-6.25°, 41.5°/-8.0°). All are edge points near the border — negligible impact.
- **5 IVT grid points:** Timed out over ocean. Negligible impact on 1,500-point grid.
