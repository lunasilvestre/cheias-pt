# Visual Hierarchy Patterns

**Principle:** Visual weight determines what users see first. Control hierarchy through color, size, position, and motion.

---

## Color Systems

### Categorical Data (Distinct Classes)

**When to use:** Fixed number of categories with no inherent order (species types, land use classes, warning types)

**Vizzuality biodiversity palette:**
```scss
$mammals: #8b62e9;       // Medium purple
$birds: #fcc44a;         // Saffron mango
$amphibians: #34bd92;    // Shamrock green
$reptiles: #92eb57;      // Screamin' green
$fishes: #38e5db;        // Cyan
```

**Pattern:** Assign semantic colors
